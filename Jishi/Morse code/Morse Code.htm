History of Morse code

In 1836, Samuel Morse demonstrated the ability of a telegraph system to send/transmit information on wires. The information was sent as a series of electric signals (like a code). Short signals are referred to as dits (they look like a dot.) and longer signals are called dahs(they look like a dash -). Soon after this an international version of Morse code became widely used. Medical uses for Morse code allow the patient to tap or blink out a message. Did you know experienced Morse code operators can enter text mush faster than someone typing using a normal keyboard. They can also do it without looking at the keyboard.

Samuel Finley Breese Morse

Samuel Finley Breese Morse, (1791-1872) was an American Painter and inventor. Samuel graduated from Yale university in 1810 and then studied painting in England. In 1815 he took up portrait painting and was very successful. In 1827 Samuel became interested in electricity. In 1844 he demonstrated to the public the practicality of the telegraph by transmitting the famous message, What had God wrote over a wire from Washington to Baltimore.

Jean-Dominique Bauby

Jean Bauby was a great man. Unfortunately he was once in coma and could only blink. With the help of his partner he created a book just by blinking his eyes making morse code letters and numbers. The book is called the Diving Bell and the Butterfly. It took them ten months and they worked four hours a day. This is a fabulous example of the capabilities of Morse code.

Benefits of Morse Code

-Only trained individuals can understand Morse code so sending a message will be safe and secure

-It is cheap to set up the equipment needed to write Morse code

-It is an like international language

-Morse code can be used in many different ways e.g. light signals

-You communicate using Morse code from far distances (if you are using a light, telegraph, etc.)

-It is used for military and medical purposes.

Disadvantages of Morse code

-It takes a lot of time to send, receive and translate Morse code

-It is easy to misread the messages

-It is slow compared to talking over a telephone for example P in Morse code is .---.

-You have to say everything letter by letter instead of word by word

-It has longer phrases
